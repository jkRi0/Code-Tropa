# C++ Challenge Mode - Difficulty Differentiation Review

## Differentiation Pattern (Based on Java Levels):

### **Easy Level:**
- Basic structure and syntax
- Simple operations (print, basic display)
- No or minimal calculations
- No complex logic

### **Average Level:**
- Adds variables and data types
- Basic calculations and operations
- Simple conditionals (if-else)
- Basic formatting

### **Difficult Level:**
- Multiple variables/arrays
- Complex calculations
- Loops and nested conditionals
- Multiple operations combined
- Advanced logic
